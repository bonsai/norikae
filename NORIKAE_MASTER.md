# NORIKAE MASTER - 東京メトロ乗り換え4択クイズ

子ども向けモバイルクイズアプリ。東京メトロの乗り換え・他社線直通をゲーム感覚で学べます。

## ファイル構成

| ファイル | 内容 |
|---|---|
| `quizData.ts` | クイズ問題データ（全18問、TypeScript） |
| `Quiz.tsx` | Reactクイズコンポーネント（モバイル子ども向けUI） |

## クイズ仕様

- **全18問**（かんたん6問 / ふつう6問 / むずかしい6問）
- **テーマ**: 東京メトロ乗り換え・他社線直通・同一ホーム乗り換え
- **UI**: 路線カラー重視、モバイル縦型、framer-motionアニメーション
- **機能**: 難易度選択 → 4択クイズ → 正解演出（解説付き）→ 結果画面 → リトライ

## 問題例

```
Q: 半蔵門線に乗ったまま東急田園都市線にいける駅は？
A: 渋谷 ← 正解（同一ホーム直通）
B: 表参道
C: 押上
D: 三軒茶屋
```

## 使用技術

- React 19 + TypeScript
- framer-motion（アニメーション）
- Tailwind CSS 4（スタイリング）
- wouter（ルーティング、`/quiz` パスで表示）
